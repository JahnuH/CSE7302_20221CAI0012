# AML Project Documentation

## Introduction
This repository contains the AML (Advanced Machine Learning) project. The goal of this project is to leverage advanced machine learning techniques to solve complex problems across various domains.

## Setup Instructions

To set up the project on your local machine,
1. **Clone the repository**:
   ```bash
   git clone https://github.com/JahnuH/CSE7302_20221CAI0012.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd CSE7302_20221CAI0012
   ```
3. **Install the required dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the project, use the following command:
```bash
python main.py
```

You can also run specific scripts for data preprocessing, modeling, and evaluation. Use the following commands:
```bash
python preprocess.py
python model.py
python evaluate.py
```
